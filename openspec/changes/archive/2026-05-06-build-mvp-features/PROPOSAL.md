## What

This change implements the core features for the HourStack MVP, a simple time-tracking application for freelancers. The implementation will follow the specifications outlined in `hourstack.md`, including:

- **Project Management**: Create projects with a name, client, and hourly rate.
- **Manual Time Tracking**: Log work sessions with start/end times, project association, and a brief description.
- **Core Timer**: A simple Start/Stop timer that automatically calculates session duration.
- **Session Notes**: Ability to add a quick note to each session.
- **Simple Dashboard**: An overview of total hours per project, generated income, and recent sessions.
- **Automatic Earnings Calculation**: Automatically calculates earnings based on session duration and project rate.
- **Timer Reminders**: A basic notification for long-running timers.
- **CSV Export**: Export session data for reporting and invoicing.

## Why

The primary goal is to quickly build a functional and valuable tool for freelancers to gain clear visibility into their time and earnings. By focusing on a minimal set of core features, we can deliver a useful product in a very short timeframe (1-3 days).

This MVP approach allows us to validate the core concept and gather user feedback before investing in more complex, automated features. It prioritizes a working, useful tool over perfection, following the philosophy of "make it work, then make it smart."
